Profil

Komunitas ILC(Informatics Learning Club) merupakan komunitas belajar pemrograman web yang ada di UPN Veteran Jawa Timur. Komunitas ini didirikan pada tahun 2006 dengan jumlah anggota 163 di tahun pertama tersebut.

Kegiatan

Kegiatan ILC(Informatics Learning Club) antara lain marathon atau belajar pemrograman web, sharing session, webinar bahasa pemrograman, dan lain-lain. Pada saat marathon atau belajar pemrogramaan web terdapat 3 skema belajar yaitu belajar dasar fundamental pemrograman web kemudian memilih skema frontend atau backend. Frontend berkaitan dengan tampilan/desain web sedangkan Backend berkaitan dengan bagaimana data itu disimpan ke database dan diproses sehingga bisa ditampilkan ke website.
